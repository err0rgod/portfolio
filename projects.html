# Tokenly-Auth

Python PyPI MIT

"Database-agnostic authentication & session management for Python."

A lightweight security toolkit that centralizes password hashing, JWT issuance, and refresh-token rotation — while leaving every database and model decision to you. No lock-in to SQLModel, Pydantic, or any specific driver.

Stack
Argon2id JWT pytest
Highlights
  • Crypto utils — Argon2id hashing/verification with brute-force protection
  • Token management — JWT create + verify, access & refresh pairs
  • Session utils — secure refresh-token hashing for rotation strategies
  • Middleware — storage-agnostic rate limiting + @require_auth decorator
  • Validation — structural checks for usernames and passwords
Install
# PyPI
pip install tokenly-auth
# issue a token pair
handler = TokenHandler(SECRET_KEY=...)
tokens = handler.createJwt(sub=user_id)

# protect a route
@require_auth(jwt_handler=handler)
def dashboard(): ...

# revlimiter

Python PyPI zero-deps MIT

"Implement rate limiting in your app, seamlessly."

A type-hinted, zero-dependency rate-limiting library implementing the two classic algorithms. Use a single bucket directly, or the manager layer to juggle many clients at once.

Highlights
  • Token bucket — allows short bursts while holding a steady average rate
  • Leaky bucket — drains requests at a fixed rate to smooth bursts
  • RateLimitManager — per-client buckets across algorithms
  • Type-safe — fully hinted, no external dependencies
Install & use
pip install revlimiter
mgr = RateLimitManager(algorithm="token-bucket", capacity=10, refill_rate=2.0)
bucket = mgr.get_bucket(client_id)
if bucket.allow_request():
    handle()

# nascar

Python / Scapy MIT

"If it's on the network, it's on the radar."

A high-performance, multi-threaded network scanner. Detects live hosts on LAN and WAN using ARP for local subnets and ICMP for external ranges, then fingerprints the OS via TTL analysis.

Highlights
  • Dual-mode detection — ARP for local subnets, ICMP for external networks
  • OS fingerprinting — infers Windows vs Linux/Unix from TTL
  • Target flexibility — CIDR ranges, single IPs, or domain names
  • Multi-threaded — concurrent scanning for speed
  • Port discovery — optional common-port scan for service mapping
Usage
# local subnet, 50 threads, verbose
python start.py -n 192.168.1.0/24 -t 50 -v

# external range with port discovery
python start.py -n 10.10.10.0/24 -lm -t 20
⚠ Requires root/admin (raw sockets). Strictly for authorized auditing and education — scan only networks you own or have permission to test.

# MidAnalyzer

Python / Streamlit Docker MIT

"Browser-based PCAP analysis, without the hosted-tool limits."

A Streamlit tool for analyzing PCAP capture files in the browser, extracting structured network insights. Built to sidestep file-size caps and slow processing of hosted analyzers — run it locally via Docker with no size limits. Part of the broader "Midstalker" MITM framework.

Highlights
  • Interactive UI — Streamlit front-end for quick inspection
  • No size limits — local Docker processing beats hosted caps
  • Minimal setup — clean, modular codebase, few dependencies
Run locally (Docker)
git clone https://github.com/err0rgod/midstalker_analyzer.git
cd midstalker_analyzer
docker build -t midstalker .
docker run -p 8501:8501 midstalker
# open http://localhost:8501

# blackwidow

C++ / ATtiny85 ★ 2 MIT

"Physical bypass for Windows Defender."

A hardware HID (BadUSB-style) research device built on the ATtiny85. Emulates a keyboard to deliver a reverse-shell payload to a Windows target — an offensive-security study in physical-access attack vectors and AV evasion.

Details

  • Topics — hid, attiny85, bypass-av
⚠ Offensive-security research. Deploying HID payloads against machines you don't own or lack authorization to test is illegal. Presented for educational/defensive study only.
